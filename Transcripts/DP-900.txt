The DP-900 Microsoft Azure Data Fundamentals exam is the exam required to earn the Azure Data Fundamentals certification. This certification demonstrates that you have the foundational knowledge of core data concepts and related Microsoft Azure data services needed to work with relational, non-relational, and analytics workloads. By passing this exam, you validate your understanding of data concepts like querying, ETL, and Azure data solutions including Cosmos DB, Synapse, and Data Factory
